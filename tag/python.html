A New Start on a New Blogging Platform

In doing a fairly substantial amout of research on blogging platforms, and most importantly, for free, I came to the conclusion that it would be great if I could find a way to leverage Github Pages and use some kind of statically generated blogging platform. I found myself on this quest after watching a course on Pluralsight that talked about this very thing. This course had me looking at JavaScript based frameworks like Hexo and Docpad, but being a Python evangelist, I wanted something that leveraged my favorite scripting language. Enter Pelican.

Pelican - A Static Site Generator, Powered by Python

Oooh, I liked the sound of that. Time to spin up a new VirtualEnv and give it a go!

A simple pip install pelican will get you started. Pelican supports writing content in reStructuredText out of the box, but I prefer to use Markdown as its more widely adopted, and in my opinion, a little easier to work with. You need to install the Markdown library as well with a pip install markdown. Bootstrapping your site is a simple one line command from Pelican.

pelican-quickstart
